# Voice Commander Extension

Control your browser and play music on YouTube using voice commands.

## How to Install and Start
1.  Open your browser (Chrome, Edge, or Brave).
2.  Navigate to the extensions management page:
    -   In Chrome: `chrome://extensions/`
    -   In Edge: `edge://extensions/`
3.  Turn on **Developer mode** (usually a toggle in the top right or left).
4.  Click the **Load unpacked** button.
5.  Select this project folder (`d:\web extensions`).
6.  The extension is now installed!

## How to Use
1.  Look for the **Voice Commander** icon in your extension toolbar.
2.  Click it to open the **Side Panel**.

    -   "Open Google"
    -   "Open GitHub"

## Features
- **Voice Commands**: Uses the Web Speech API for real-time recognition.
- **Side Panel Integration**: A premium UI that stays open while you browse.
- **YouTube Automation**: Automatically searches and plays the first video.
- **Modern UI**: Dark mode with elegant waves and animations.

## Permissions Required
- `sidePanel`: To display the voice interface.
- `tabs`: To open new pages.
- `scripting`: To interact with YouTube pages (play videos).
- `activeTab`: To access the current tab when needed.
